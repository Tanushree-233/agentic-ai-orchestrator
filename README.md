Agentic AI System

Overview

This project implements a multi-agent AI system that accepts a complex task, decomposes it into steps, assigns each step to a specialized agent, streams partial outputs, handles failures gracefully, and performs manual batching.

Features

- Task decomposition
- Multiple specialized agents
- Async pipeline execution
- Streaming outputs
- Failure handling
- Manual batching

Agents

1. Planner Agent
2. Retriever Agent
3. Analyzer Agent
4. Writer Agent
